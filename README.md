Fetc APi 

Cara mengistall aplikasi ini adalah dengan mengclone folder.
lakukan perintah 'docker compose up' untuk menjalankan docker.
lalu lakukan perintah "composer install" untuk menginstal composer.
lakukan perubahn .env sesui dengan pengaturan docker yaitu menubah nama DB username dan password dan port mariaDB.
lakukan migration untuk dengan perintah 'php artisan migrate'
setelaah database terinstal lakukan perintah menjalankan 'php artisan serv' untuk menjalankan serer
akses localhost untuk menampilkan web.
